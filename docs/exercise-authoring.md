# Exercise Authoring Notes

Every new exercise should include learner-facing comments in `exercise.cpp` before the function to implement.

Use this structure:

```cpp
// Concept: The core C++ idea this exercise teaches.
// Practice: The concrete action the learner should perform.
// Watch for: The most likely mistake or edge case.
//
// Task: The exact behavior to implement.
```

Keep the comments short and specific. The goal is to make it obvious what the learner is practicing without explaining the full solution.
